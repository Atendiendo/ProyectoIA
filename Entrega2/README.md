# Requisitos:
- Se requiere una distribución de LaTeX con los paquetes estándar instalados. Se recomienda una instalación **completa** para asegurar que bibliotecas como `tikz` y `algorithm2e` estén disponibles:
    - TeX Live (Linux / macOS / Windows)
    - MiKTeX (Windows)
    - MacTeX (macOS)
- GNU Make 4.3 o superior

# Instrucciones
- Para compilar usar comando ```make```
- Para limpiar archivos usar comando ```make clean```
